What is a gitignore file and what should it include for a Java project?
A gitignore file specifies intentionally untracked files that git should ignore. File should have .log,.class for java project.

� What does fast-forward in Git mean?
when we merge the checkout(present) branch with the branch before this or ahead of this, then this is called fast forward in git


� Why should you have different branches?
Because when we make changes to the master branch and we fail, then it will have effect on previous code in master branch. So to keep the master branch safe, we create other branches and then finally merge it on to master.
